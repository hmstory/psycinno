Why Now

아직 아무도 안 하고 있다

Talkspace BetterHelp 트로스트 우리
상담 형태 1:1 텍스트/화상 1:1 텍스트/화상 1:1 텍스트 그룹 음성
매칭 방식 의사 수동 의뢰 설문 → AI 추천 사용자 직접 검색 AI 대화 → 자동 그룹
AI 역할 없음 상담사 추천만 없음 평가 + 분류 + 매칭
그룹 상담 없음 있음 (수동 배정) 없음 AI 자동 매칭

기존 서비스는 전부 1:1이고, 사람이 매칭한다.
AI가 매칭하고, 그룹으로 연결하는 서비스는 아직 없다.

Phase 1 — Now

같이 만들어봐요

그룹 매칭을 위한 EBT 기반 심리 평가 챗봇

📚
심리학 기반 AI 수업
전용 플랫폼에서 Prompt Engineering으로 개발. RAG에 근거기반치료(EBT)가 내장되어 있어 대화 속에 임상 척도를 자연스럽게 녹입니다.
📊
구조화된 결과 출력
대화가 끝나면 고민 카테고리, 심각도, 호소문제 요약이 자동 생성됩니다. 이 결과가 2단계 그루핑의 입력값이 됩니다.
🛡️
안전장치 설계
"본 서비스는 위기 상담을 대체하지 않습니다." 사전 동의 + 위기 연계처(1393) 상시 표시.
챗봇 대화
"요즘 힘들어요..."
→
평가 결과
카테고리 + 심각도
→
