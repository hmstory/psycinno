# Clarify v2: sub agent 피드백 기반 (2026-03-08)

## Before (원본)
> "AI 대화로 고민을 파악하고, 비슷한 사람끼리 그룹 매칭해서, 자격 있는 상담사가 이끄는 음성 그룹 상담을 제공하는 앱. 5분 대화로 설문 없이 그루핑."

## After (구체화됨)

**목표**: EBT 기반 대화형 심리 평가 -> AI 그루핑 -> 상담사 음성 그룹 세션
**대상 사용자**: 일반 성인 (스트레스, 사회적 고립감)
**수업 연결**: 조별 과제 주제로 제안 -> 1단계 챗봇을 전용 플랫폼에서 개발

##  피드백 원문 요약 (시뮬레이션)

1. **EBT 없음** -- 슬라이드 어디에도 근거기반치료 명시 없음. 수업 필수 요건
2. **5분은 너무 짧다** -- 임상적 근거 부족. 표준화 도구(PHQ-9, GAD-7)를 대화에 녹이는 게 타당
3. **위기 대응 없음** -- AI가 위험 사용자를 놓치면? 안전장치 필요
4. **연구 질문 없음** -- 서비스 피칭이지 학술 프로젝트가 아님
5. **경쟁 분석 없음** -- Talkspace Groups 등과 차별점 불명확
6. **긍정 피드백** -- "AI를 치료가 아니라 연결에 쓴다"는 관점은 신선. DMH 랩에서 안 해본 것
