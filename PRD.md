# 보이스 그룹 카운슬링 앱

## 문제

> 사회적 고립감과 스트레스를 겪는 성인들이 심리 지원을 받고 싶지만, 1:1 상담은 비용과 접근성 문제로 어렵다. 기존 서비스(Talkspace, BetterHelp, 트로스트)는 전부 1:1 중심이고, 그룹 매칭은 사람이 수동으로 한다. AI가 대화를 통해 고민을 파악하고, 비슷한 사람끼리 자동으로 그룹을 구성하여 자격 있는 상담사가 이끄는 음성 그룹 세션을 지원한다.

- **현재 상태**: 심리 상담 앱은 대부분 1:1 텍스트/화상 기반. 그룹 상담이 있어도 설문 기반 수동 배정. AI는 상담사 추천에만 사용됨.
- **원하는 상태**: 사용자가 앱에 접속해 AI와 대화하면, EBT 기반 척도로 고민 주제와 심각도를 평가하고, 비슷한 사람끼리 소그룹(4~5명)을 자동 구성. 자격 있는 상담사가 음성으로 그룹 세션을 진행.
- **왜 보이스인가**: 얼굴 부담 없이, 밤늦은 시간에도, 어디서든 편하게 참여 가능.

## 경쟁 분석

| | Talkspace | BetterHelp | 트로스트 | 우리 |
|--|-----------|------------|---------|------|
| 상담 형태 | 1:1 텍스트/화상 | 1:1 텍스트/화상 | 1:1 텍스트 | **그룹 음성** |
| 매칭 방식 | 의사 수동 의뢰 | 설문 -> AI 추천 | 사용자 직접 검색 | **AI 대화 -> 자동 그룹** |
| AI 역할 | 없음 | 상담사 추천만 | 없음 | **평가 + 분류 + 매칭** |
| 그룹 상담 | 없음 | 있음 (수동 배정) | 없음 | **AI 자동 매칭** |

**차별점**: AI가 매칭하고, 그룹으로 연결하는 서비스는 아직 없다.

## 3단계 로드맵

| Phase | 내용 | 개발 방식 |
|-------|------|----------|
| **1단계** | EBT 기반 대화형 심리 평가 챗봇 | 수업 전용 플랫폼 (Prompt Engineering + RAG) |
| **2단계** | AI 그루핑 엔진 (평가 결과 -> 자동 매칭) | 자체 개발 |
| **3단계** | 상담사가 이끄는 소그룹 음성 집단 상담 | 자체 개발 (엔드 픽쳐) |
