# 보이스 그룹 카운슬링 앱

## 문제

> 사회적 고립감과 스트레스를 겪는 성인들이 심리 지원을 받고 싶지만, 1:1 상담은 비용·접근성 문제로 어렵다. AI가 대화를 분석해 비슷한 고민을 가진 사람끼리 자동으로 그룹을 구성하고, 음성 그룹 세션을 지원한다.

- **현재 상태**: 심리 상담 앱은 대부분 1:1 텍스트 기반이거나 사전 설문으로 그룹을 수동 배정함. 사용자가 직접 그룹을 찾아야 하며 평균 대기 시간 1~2주.
- **원하는 상태**: 사용자가 앱에 접속해 간단한 음성 대화를 하면, AI가 감정 상태와 고민 주제를 탐지해 1분 내로 적합한 그룹(3~5명)에 자동 배정. 그룹 세션 시작.
- **성공 기준**:
  - 첫 대화 후 그룹 배정까지 1분 이내
  - 세션 참여자 만족도 4점 이상 (5점 만점)
